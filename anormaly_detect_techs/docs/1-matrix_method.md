行列分解を用いる異常検知手法の中で、現在「最新かつ高性能」とされるアプローチは、従来の数学的な行列分解にディープラーニングの柔軟性を組み合わせた **「ディープ行列分解（Deep Matrix Factorization）」** や、テンソル（多次元行列）に拡張した **「テンソル堅牢主成分分析（Tensor RPCA）」** です。

単なる線形な分解ではなく、複雑な非線形パターンやデータの高次元構造を捉えられる点が強みです。


### 1. Deep Matrix Factorization (Deep MF)

従来のNMF（非負値行列分解）などを多層化（Deep化）した手法です。

* **特徴:** 1層の分解では捉えきれない、画像の潜在的な「抽象度の高い特徴」を階層的に抽出します。
* **性能が高い理由:** ニューラルネットワークのように層を重ねることで、画像のテクスチャ、形状、構造といった異なるスケールの情報を同時に分解・保持できます。
* **最新の動向:** グラフ構造を組み込んだ「Graph Regularized Deep MF」などが、複雑な背景を持つ画像からの異常抽出に高い性能を示しています。

### 2. Tensor Robust PCA (TRPCA)

画像を単なる行列（2次元）としてではなく、カラーチャンネルや時間軸（動画の場合）を含めた「テンソル（多次元）」として直接分解する手法です。

* **特徴:** 行列分解（RPCA）を3次元以上に拡張したもので、空間的なつながり（縦・横）と色の相関を壊さずに分解できます。
* **性能が高い理由:** 従来のRPCAでは画像をベクトル化（1次元化）する必要があり、近傍ピクセルの関係性が失われがちでしたが、TRPCAはこれを維持するため、**微細な点キズや糸くずのような異常**を極めて正確に「Sparse（スパース）成分」として分離できます。

### 3. Learned Robust Matrix Completion (LRMC) / Deep Unfolding

「行列分解のアルゴリズム」そのものをニューラルネットワークの層として展開（Unfolding）した最新手法です。

* **特徴:** 数学的な反復計算（最適化アルゴリズム）を、学習可能なネットワークとして構築します。
* **性能が高い理由:** 従来の行列分解は「パラメータ調整」が非常に難しかったのですが、この手法はデータから最適な分解パラメータを自動で学習します。2024年〜2025年の最新研究でも、**動画の背景差分や霧除去、医療画像**などでSOTA（最高精度）を記録しています。


### 手法の比較

| 手法 | 主な用途 | 強み |
| --- | --- | --- |
| **Deep MF** | 複雑なテクスチャの画像 | 階層的な特徴抽出により、高度な異常を捉える |
| **Tensor RPCA** | カラー画像・動画・3Dデータ | 空間・時間情報を維持し、ノイズと異常を鋭く分離 |
| **LRMC (Deep Unfolding)** | 高速かつ高精度な実務 | 数学的厳密さとAIの学習能力を両立 |

### 次のステップ：どれを選ぶべきか？

* **「動画」や「カラーの塊」から異常を見つけたい**なら、**Tensor RPCA**（またはそのライブラリ）を調べるのが最も近道です。
* **「静止画の非常に複雑な模様」**の中から異常を探したいなら、**Deep MF** のアプローチが適しています。
