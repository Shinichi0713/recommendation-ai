## 濃度とは

集合論における**濃度（cardinality）** は、集合の「大きさ」や「要素の個数」を表す概念です。有限集合なら「要素の個数」と同じですが、無限集合にも拡張された抽象的な「大きさ」の指標です。



### 1. 直観的なイメージ
- 有限集合：  
  「集合 A の濃度」は、A に含まれる要素の個数です。  
  例：A = {1, 2, 3} なら |A| = 3。
- 無限集合：  
  自然数全体の集合 ℕ と、整数全体の集合 ℤ はどちらも無限個の要素を持ちますが、  
  「同じ種類の無限」かどうかを比較するために濃度を使います。


### 2. 濃度の厳密な定義（同値関係として）
集合 A, B の濃度が等しい（|A| = |B|）とは、  
**A から B への全単射（一対一対応）が存在する**ことと定義されます。

- 全単射とは：
  - 単射：異なる要素は異なる行き先へ（一対一）
  - 全射：B のどの要素にも A の要素が対応（漏れなし）

この「全単射が存在する」という関係は同値関係であり、その同値類のことを**濃度**と呼びます。


### 3. 有限集合の濃度
有限集合 A に対しては、濃度 |A| は単に「要素の個数」です。

- A = {a, b, c} → |A| = 3
- B = {1, 2, 3, 4} → |B| = 4

有限集合の濃度が等しいとは、**要素の個数が同じ**ということです。


### 4. 無限集合の濃度
無限集合にも濃度が定義され、代表的なものに次のようなものがあります。

#### (1) 可算無限（countably infinite）
- 自然数全体 ℕ = {1, 2, 3, …} の濃度を **ℵ₀**（アレフ・ゼロ）と書きます。
- ℕ と全単射が作れる集合は、すべて濃度 ℵ₀ を持つと定義します。
  - 例：整数全体 ℤ、有理数全体 ℚ は ℵ₀ の濃度です。

#### (2) 非可算無限（uncountably infinite）
- 実数全体 ℝ の濃度は ℵ₀ より**真に大きい**ことが知られています（カントールの対角線論法）。
- この濃度を **𝔠**（continuum）と書きます。
- さらに、任意の集合 A に対して、そのべき集合 P(A) の濃度は A の濃度より真に大きい（カントールの定理）ので、  
  「無限の大きさ」はいくらでも大きいものが存在します。


### 5. 濃度の比較
集合 A, B について、  
- |A| ≤ |B| とは、**A から B への単射が存在する**ことと定義します。
- |A| = |B| とは、全単射が存在すること。
- |A| < |B| とは、|A| ≤ |B| だが |A| ≠ |B| であること。

これにより、有限・無限を問わず「大きさ」を比較できます。

### 6. 重要な性質
- ベルンシュタインの定理：  
  |A| ≤ |B| かつ |B| ≤ |A| ならば |A| = |B| が成り立ちます。  
  つまり、「A から B への単射」と「B から A への単射」が両方あれば、全単射が存在する（濃度が等しい）ことが保証されます。
- 選択公理を仮定すると、任意の集合の濃度は比較可能（|A| ≤ |B| か |B| ≤ |A| のどちらかが成り立つ）と示せます。

---

__証明__

ベルンシュタインの定理（シュレーダー＝ベルンシュタインの定理）の証明を、できるだけ直観的に説明します。

__定理の再確認__

- 集合 A, B に対し、  
  |A| ≤ |B| かつ |B| ≤ |A| ならば |A| = |B|。
- 言い換えると：
  - f: A → B が単射
  - g: B → A が単射  
  が与えられたとき、A から B への全単射が存在する。

__証明の方針__

A の要素を「g を何回も使って B から戻ってきたもの」と「そうでないもの」に分け、  
その分類に応じて f と g の逆写像をうまく組み合わせて、A から B への全単射を構成します。

__1. 記号の準備__

- f: A → B を単射
- g: B → A を単射

ここで、A の部分集合を次のように定義します。

- A₀ = A
- A₁ = g(B) ⊆ A（B の g による像）
- A₂ = g(f(A₀)) ⊆ A
- A₃ = g(f(A₁)) ⊆ A
- A₄ = g(f(A₂)) ⊆ A
- …

一般に、n ≥ 0 に対して
- A₀ = A
- Aₙ₊₁ = g(f(Aₙ))

と帰納的に定義します。

__2. A の分割__

この列 A₀, A₁, A₂, … は
- A₀ ⊇ A₁ ⊇ A₂ ⊇ …  
という**減少列**になっています（包含関係が逆転しない）。

そこで、差集合を取って A を互いに素な部分に分けます。

- C₀ = A₀ \ A₁
- C₁ = A₁ \ A₂
- C₂ = A₂ \ A₃
- …
- Cₙ = Aₙ \ Aₙ₊₁
- C_∞ = A₀ ∩ A₁ ∩ A₂ ∩ …（すべての Aₙ の共通部分）

すると、A は
- A = C₀ ∪ C₁ ∪ C₂ ∪ … ∪ C_∞  
と**互いに素な和集合**に分解されます。

__3. B 側の対応する部分__

B 側も同様に分割します。

- B₀ = B
- B₁ = f(A₀) ⊆ B
- B₂ = f(A₁) ⊆ B
- B₃ = f(A₂) ⊆ B
- …

ここで、B の部分集合 Dₙ を
- D₀ = B₀ \ B₁
- D₁ = B₁ \ B₂
- D₂ = B₂ \ B₃
- …
- D_∞ = B₀ ∩ B₁ ∩ B₂ ∩ …

と定義すると、B は
- B = D₀ ∪ D₁ ∪ D₂ ∪ … ∪ D_∞  
と互いに素な和集合に分解されます。

__4. 全単射の構成__

A の各成分 Cₙ と B の各成分 Dₙ の対応を次のように定義します。

__(1) 有限ステップの部分（Cₙ, n 有限）__

- C₀ の各元 a ∈ C₀ に対して、f(a) ∈ D₁ を対応させる。
- C₁ の各元 a ∈ C₁ に対して、f(a) ∈ D₂ を対応させる。
- 一般に、a ∈ Cₙ に対して f(a) ∈ Dₙ₊₁ を対応させる。

ここで重要なのは：
- f は単射なので、Cₙ から Dₙ₊₁ への写像は単射。
- さらに、Dₙ₊₁ の任意の元 b は、ある a ∈ Aₙ に対して b = f(a) と書け、  
  a が Aₙ₊₁ に属さない（つまり a ∈ Cₙ）ことから、この対応は全射にもなります。

したがって、各 n について
- Cₙ → Dₙ₊₁ は全単射（f の制限として実現できる）。

__(2) 無限ステップの部分（C_∞）__

- C_∞ の元 a は、すべての n について a ∈ Aₙ を満たします。
- このとき、g の逆写像 g⁻¹: g(B) → B を考え、  
  a ∈ C_∞ に対して g⁻¹(a) ∈ D_∞ を対応させます。

ここで：
- g は単射なので、g(B) 上では逆写像 g⁻¹ が定義でき、  
  g⁻¹: g(B) → B は全単射です。
- C_∞ ⊆ g(B) なので、g⁻¹ の C_∞ への制限は C_∞ → D_∞ の全単射を与えます。

__5. 全体としての写像 h: A → B__

以上をまとめて、写像 h: A → B を次のように定義します。

- a ∈ Cₙ（n 有限）のとき：h(a) = f(a) ∈ Dₙ₊₁
- a ∈ C_∞ のとき：h(a) = g⁻¹(a) ∈ D_∞

この h は：
- A の互いに素な分割 C₀, C₁, …, C_∞ のそれぞれで全単射を定め、
- それらの像も B の互いに素な分割 D₁, D₂, …, D_∞ になっているので、

全体として **h: A → B は全単射**になります。

__6. 結論__

単射 f: A → B と単射 g: B → A が存在するとき、上のように A を分割し、  
f と g⁻¹ を組み合わせることで A から B への全単射 h を構成できます。

したがって、
- |A| ≤ |B| かつ |B| ≤ |A| ⇒ |A| = |B|
が成り立ちます。

__補足__

- この証明は「背理法」を使わず、**具体的に全単射を構成する**形になっています。
- 実は、この構成は「g を繰り返し適用して A に戻れる回数」に応じて A を分類している、と解釈できます。
- 選択公理は使っていません（f, g が与えられているので、それらから具体的に h を作れる）。

---


### 7. 総括
- 濃度は「集合の大きさ」を表す抽象的な概念。
- 有限集合では「要素の個数」と同じ。
- 無限集合では「全単射が存在するかどうか」で大きさを比較する。
- 可算無限（ℵ₀）と非可算無限（𝔠 など）があり、無限にも大小関係がある。
