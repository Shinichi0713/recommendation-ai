2次のラグランジュ関数を最適化する問題は、一般に「2次計画問題（Quadratic Programming, QP）」にラグランジュの未定乗数法を適用した形として扱えます。

以下では、次のような標準的な2次計画問題を考えます。

  • 目的関数（2次）：

    f(x)=12xTQx+cTxf(\mathbf{x}) = \frac{1}{2} \mathbf{x}^T Q \mathbf{x} + \mathbf{c}^T \mathbf{x} f(x)=21​xTQx+cTx

    ここで QQQ は対称行列（正定値とは限らない）とします。

  • 制約条件（線形）：

    Ax=bA \mathbf{x} = \mathbf{b} Ax=b

    あるいは

    Ax≤bA \mathbf{x} \leq \mathbf{b} Ax≤b

    など。

1. 等式制約付き2次計画問題（ラグランジュ関数の構成）

等式制約

Ax=bA \mathbf{x} = \mathbf{b} Ax=b

のもとで f(x)f(\mathbf{x})f(x) を最小化する問題を考えます。

ラグランジュ関数は

L(x,λ)=12xTQx+cTx+λT(Ax−b)\mathcal{L}(\mathbf{x}, \boldsymbol{\lambda}) = \frac{1}{2} \mathbf{x}^T Q \mathbf{x} + \mathbf{c}^T \mathbf{x} + \boldsymbol{\lambda}^T (A \mathbf{x} - \mathbf{b}) L(x,λ)=21​xTQx+cTx+λT(Ax−b)

と書けます。ここで λ\boldsymbol{\lambda}λ はラグランジュ乗数ベクトルです。

2. 最適性条件（KKT条件）

最適解 (x∗,λ∗)(\mathbf{x}^*, \boldsymbol{\lambda}^*)(x∗,λ∗) は、次のKKT条件を満たします。

  1. 勾配条件（1階必要条件）

    ∇xL=Qx∗+c+ATλ∗=0\nabla_{\mathbf{x}} \mathcal{L} = Q \mathbf{x}^* + \mathbf{c} + A^T \boldsymbol{\lambda}^* = 0 ∇x​L=Qx∗+c+ATλ∗=0

    ∇λL=Ax∗−b=0\nabla_{\boldsymbol{\lambda}} \mathcal{L} = A \mathbf{x}^* - \mathbf{b} = 0 ∇λ​L=Ax∗−b=0

  2. 2階条件（ヘッセ行列の正定値性） 制約を満たす方向（Ad=0A \mathbf{d} = 0Ad=0 を満たす任意の d≠0\mathbf{d} \neq 0d=0）について、

    dTQd≥0\mathbf{d}^T Q \mathbf{d} \ge 0 dTQd≥0

    が成り立つならば、局所最適解となります（十分条件）。

3. 最適解の導出（KKT系の解法）

KKT条件は線形方程式系に帰着します：

[QATA0][x∗λ∗]=[−cb]\begin{bmatrix} Q & A^T \\ A & 0 \end{bmatrix} \begin{bmatrix} \mathbf{x}^* \\ \boldsymbol{\lambda}^* \end{bmatrix} = \begin{bmatrix} -\mathbf{c} \\ \mathbf{b} \end{bmatrix} [QA​AT0​][x∗λ∗​]=[−cb​]

この系を解くことで、最適解 (x∗,λ∗)(\mathbf{x}^*, \boldsymbol{\lambda}^*)(x∗,λ∗) が得られます。

  • QQQ が正定値で、かつ AAA がフルランクならば、このKKT行列は正則であり、解は一意に定まります。
  • 数値的には、コレスキー分解やLDL分解を用いた直接法、あるいは反復法（共役勾配法など）で解くことができます。

4. 不等式制約付き2次計画問題の場合

制約が

Ax≤bA \mathbf{x} \leq \mathbf{b} Ax≤b

のような不等式の場合、ラグランジュ関数は

L(x,λ)=12xTQx+cTx+λT(Ax−b)\mathcal{L}(\mathbf{x}, \boldsymbol{\lambda}) = \frac{1}{2} \mathbf{x}^T Q \mathbf{x} + \mathbf{c}^T \mathbf{x} + \boldsymbol{\lambda}^T (A \mathbf{x} - \mathbf{b}) L(x,λ)=21​xTQx+cTx+λT(Ax−b)

と同様に定義されますが、KKT条件には相補性条件が加わります：

  • 主実行可能：

    Ax∗≤bA \mathbf{x}^* \leq \mathbf{b} Ax∗≤b

  • 双対実行可能：

    λ∗≥0\boldsymbol{\lambda}^* \ge 0 λ∗≥0

  • 勾配条件：

    Qx∗+c+ATλ∗=0Q \mathbf{x}^* + \mathbf{c} + A^T \boldsymbol{\lambda}^* = 0 Qx∗+c+ATλ∗=0

  • 相補性条件：

    λi∗(Aix∗−bi)=0,i=1,…,m\lambda_i^* (A_i \mathbf{x}^* - b_i) = 0,\quad i = 1, \dots, m λi∗​(Ai​x∗−bi​)=0,i=1,…,m

この場合、最適解を解析的に求めるのは一般に困難であり、アクティブセット法や内点法などの数値解法を用いて解くことになります。

5. まとめ

  • 2次のラグランジュ関数の最適化は、2次計画問題の枠組みで扱えます。
  • 等式制約の場合は、KKT条件から線形方程式系を解くことで最適解が得られます。
  • 不等式制約の場合は、KKT条件に相補性条件が加わり、数値解法（アクティブセット法、内点法など）が必要になります。

もし具体的な行列 Q,AQ, AQ,A やベクトル c,b\mathbf{c}, \mathbf{b}c,b が与えられていれば、それらを代入してKKT系を書き下し、数値的に解く手順も示せます。
