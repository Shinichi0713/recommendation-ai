    双対変数とは

    最適化における双対変数（dual variable） は、主問題（primal problem） の制約条件に対応して現れる、双対問題（dual problem） における変数のことです。

    1. 双対変数の位置づけ

    多くの最適化問題（線形計画、凸最適化など）では、

    • 元の問題を「主問題」
    • それから導かれる別の最適化問題を「双対問題」

    として扱います。

    • 主問題の変数を主変数（primal variables）
    • 双対問題の変数を双対変数（dual variables）

    と呼びます。

    特に、主問題が

    min⁡xf(x)s.t.gi(x)≤0,i=1,…,mhj(x)=0,j=1,…,p\begin{aligned} \min_x &\quad f(x) \\ \text{s.t.} &\quad g_i(x) \le 0,\quad i=1,\dots,m \\ &\quad h_j(x) = 0,\quad j=1,\dots,p \end{aligned} xmin​s.t.​f(x)gi​(x)≤0,i=1,…,mhj​(x)=0,j=1,…,p​

    のような形のとき、双対問題はラグランジュ関数

    L(x,λ,ν)=f(x)+∑i=1mλigi(x)+∑j=1pνjhj(x)L(x,\lambda,\nu) = f(x) + \sum_{i=1}^m \lambda_i g_i(x) + \sum_{j=1}^p \nu_j h_j(x) L(x,λ,ν)=f(x)+i=1∑m​λi​gi​(x)+j=1∑p​νj​hj​(x)

    から定義され、そのとき

    • λi\lambda_iλi​：不等式制約 gi(x)≤0g_i(x) \le 0gi​(x)≤0 に対応する双対変数
    • νj\nu_jνj​：等式制約 hj(x)=0h_j(x) = 0hj​(x)=0 に対応する双対変数

    となります。

    2. 双対変数の意味（経済的・物理的な解釈）

    双対変数は、しばしばシャドウプライス（shadow price） や感度係数として解釈されます。

    (1) 資源制約の「価格」としての解釈（線形計画など）

    例えば、資源制約付きの利益最大化問題で

    max⁡c⊤xs.t.Ax≤b,x≥0\begin{aligned} \max &\quad c^\top x \\ \text{s.t.} &\quad Ax \le b,\quad x \ge 0 \end{aligned} maxs.t.​c⊤xAx≤b,x≥0​

    という主問題を考えます。ここで

    • bbb：利用可能な資源量（在庫、時間、予算など）
    • AAA：各活動 xjx_jxj​ がどれだけ資源を消費するか

    とすると、双対変数（この場合は双対問題の変数 yyy）は

    「資源 bib_ibi​ を 1 単位だけ追加したとき、最適目的関数値（利益）がどれだけ増えるか」

    を表す限界価格と解釈できます。

    • 双対変数が大きい制約：その資源を少し増やすと目的関数が大きく改善する → その資源は「希少」で価値が高い
    • 双対変数が 0 の制約：その資源を増やしても目的関数は変わらない → 余剰資源（スラックがある）

    という経済的な意味を持ちます。

    (2) 感度としての解釈（一般の最適化）

    より一般的には、双対変数は

    「制約条件をわずかに緩めたとき、最適値がどれだけ変化するか」

    という感度（sensitivity） を表します。

    • 不等式制約 gi(x)≤0g_i(x) \le 0gi​(x)≤0 の双対変数 λi\lambda_iλi​：
      • 制約の右辺を少し緩める（例：gi(x)≤εg_i(x) \le \varepsilongi​(x)≤ε）と、最適値が約 λi⋅ε\lambda_i \cdot \varepsilonλi​⋅ε だけ改善（または悪化）する
    • 等式制約 hj(x)=0h_j(x) = 0hj​(x)=0 の双対変数 νj\nu_jνj​：
      • 右辺を少しずらす（例：hj(x)=δh_j(x) = \deltahj​(x)=δ）と、最適値が約 νj⋅δ\nu_j \cdot \deltaνj​⋅δ だけ変化する

    このように、双対変数は「制約の厳しさが目的関数に与える影響の強さ」を数値化したものと見なせます。

    3. 双対変数の性質（KKT条件との関係）

    凸最適化などでよく使われるKKT条件（Karush–Kuhn–Tucker条件） では、最適解において

    1. 主実行可能性：制約を満たす
    2. 双対実行可能性：λ≥0\lambda \ge 0λ≥0
    3. 相補性条件：λigi(x∗)=0\lambda_i g_i(x^*) = 0λi​gi​(x∗)=0

    などが成り立ちます。

    • 相補性条件 λigi(x∗)=0\lambda_i g_i(x^*) = 0λi​gi​(x∗)=0 は、
      • もし制約が「タイト（有効）」なら gi(x∗)=0g_i(x^*) = 0gi​(x∗)=0 で、λi\lambda_iλi​ は 0 でも正でもよい
      • もし制約が「非タイト（緩い）」なら gi(x∗)<0g_i(x^*) < 0gi​(x∗)<0 で、λi=0\lambda_i = 0λi​=0 となる
    • つまり、緩い制約に対応する双対変数は 0になり、その制約は最適値に影響しないことを意味します。

    4. 双対変数が資源の限界価格になる理由

    1. 主問題と双対問題の設定

    主問題（利益最大化）：

    max⁡xc⊤xs.t.Ax≤b,x≥0\begin{aligned} \max_x &\quad c^\top x \\ \text{s.t.} &\quad Ax \le b,\quad x \ge 0 \end{aligned} xmax​s.t.​c⊤xAx≤b,x≥0​

    ここで

    • xxx：活動量（生産量など）
    • ccc：各活動の単位利益
    • AAA：各活動が資源をどれだけ消費するか
    • bbb：利用可能な資源量

    とします。

    この主問題に対応する双対問題は

    min⁡yb⊤ys.t.A⊤y≥c,y≥0\begin{aligned} \min_y &\quad b^\top y \\ \text{s.t.} &\quad A^\top y \ge c,\quad y \ge 0 \end{aligned} ymin​s.t.​b⊤yA⊤y≥c,y≥0​

    となります。ここで yyy が双対変数です。

    2. 強双対定理と最適値の関係

    線形計画では、主問題と双対問題がともに実行可能ならば、強双対定理により

    max⁡xc⊤x=min⁡yb⊤y\max_{x} c^\top x = \min_{y} b^\top y xmax​c⊤x=ymin​b⊤y

    が成り立ちます。

    つまり、最適目的関数値（最適利益）を p∗p^*p∗ とすると

    p∗=c⊤x∗=b⊤y∗p^* = c^\top x^* = b^\top y^* p∗=c⊤x∗=b⊤y∗

    が成り立ちます。

    ここで

    • x∗x^*x∗：主問題の最適解
    • y∗y^*y∗：双対問題の最適解（双対変数の最適値）

    です。

    3. 資源量 bbb を少し変えたときの変化

    資源ベクトル bbb を少しだけ変えたときの最適値の変化を考えます。

    資源 iii の量 bib_ibi​ を bi+Δbib_i + \Delta b_ibi​+Δbi​ に変えたときの新しい最適値を p∗(bi+Δbi)p^*(b_i + \Delta b_i)p∗(bi​+Δbi​) とします。

    双対問題の形

    min⁡yb⊤ys.t. A⊤y≥c, y≥0\min_y b^\top y \quad \text{s.t. } A^\top y \ge c,\ y \ge 0 ymin​b⊤ys.t. A⊤y≥c, y≥0

    を見ると、目的関数は b⊤yb^\top yb⊤y です。
    もし最適双対解 y∗y^*y∗ が変わらないと仮定すると（少なくとも小さな変化では近似として成り立ちます）、

    p∗(bi+Δbi)≈b⊤y∗+yi∗Δbip^*(b_i + \Delta b_i) \approx b^\top y^* + y_i^* \Delta b_i p∗(bi​+Δbi​)≈b⊤y∗+yi∗​Δbi​

    となります。なぜなら、bbb の iii 番目の成分だけが Δbi\Delta b_iΔbi​ 増えると、目的関数は

    (b+Δbiei)⊤y∗=b⊤y∗+yi∗Δbi(b + \Delta b_i e_i)^\top y^* = b^\top y^* + y_i^* \Delta b_i (b+Δbi​ei​)⊤y∗=b⊤y∗+yi∗​Δbi​

    となるからです。

    したがって、最適値の変化分は

    Δp∗≈yi∗Δbi\Delta p^* \approx y_i^* \Delta b_i Δp∗≈yi∗​Δbi​

    となり、単位当たりの変化は

    Δp∗Δbi≈yi∗\frac{\Delta p^*}{\Delta b_i} \approx y_i^* Δbi​Δp∗​≈yi∗​

    です。

    4. 限界価格としての解釈

    以上から、

    • 資源 iii の量 bib_ibi​ を 1 単位増やしたとき（Δbi=1\Delta b_i = 1Δbi​=1）
    • 最適利益は約 yi∗y_i^*yi∗​ だけ増える

    と解釈できます。

    つまり、

    双対変数 yi∗y_i^*yi∗​ は、「資源 iii を 1 単位追加したときの最適目的関数値の増分」を表す**限界価格（シャドウプライス）**です。

    • yi∗>0y_i^* > 0yi∗​>0：その資源は希少で、少し増やすと利益が大きく増える
    • yi∗=0y_i^* = 0yi∗​=0：その資源は余っており、増やしても利益は変わらない（スラックがある）

    という経済的意味を持ちます。

    5. 直感的なイメージ

    もう少し直感的に言うと：

    • 主問題：資源 bbb の制約のもとで利益 c⊤xc^\top xc⊤x を最大化
    • 双対問題：資源の「価格」yyy を決めて、資源の総価値 b⊤yb^\top yb⊤y を最小化（＝利益最大化と釣り合うように価格を決める）

    最適なときには、各資源の「価格」yi∗y_i^*yi∗​ が、その資源を 1 単位増やしたときの利益の増加分と一致します。
    これが「双対変数＝限界価格」という解釈の根拠です。

    まとめると：

    • 線形計画の強双対定理により p∗=b⊤y∗p^* = b^\top y^*p∗=b⊤y∗
    • 資源 bib_ibi​ を少し変えたときの最適値の変化は、双対変数 yi∗y_i^*yi∗​ に比例する
    • よって yi∗y_i^*yi∗​ は「資源 iii を 1 単位追加したときの最適利益の増分」＝限界価格と解釈できる

    ということから説明できます。

    双対変数が存在するメリット

    各制約に対応する双対変数（ラグランジュ乗数）をラグランジュ関数に含めるか・含めないかで、何が変わるのか？について説明していきます。

    1. ラグランジュ関数の標準的な定義

    一般に、制約付き最適化問題

    min⁡xf(x)s.t.gi(x)≤0,i=1,…,mhj(x)=0,j=1,…,p\begin{aligned} \min_x &\quad f(x) \\ \text{s.t.} &\quad g_i(x) \le 0,\quad i=1,\dots,m \\ &\quad h_j(x) = 0,\quad j=1,\dots,p \end{aligned} xmin​s.t.​f(x)gi​(x)≤0,i=1,…,mhj​(x)=0,j=1,…,p​

    に対して、ラグランジュ関数は

    L(x,λ,ν)=f(x)+∑i=1mλigi(x)+∑j=1pνjhj(x)L(x,\lambda,\nu) = f(x) + \sum_{i=1}^m \lambda_i g_i(x) + \sum_{j=1}^p \nu_j h_j(x) L(x,λ,ν)=f(x)+i=1∑m​λi​gi​(x)+j=1∑p​νj​hj​(x)

    と定義されます。

    ここで

    • λi\lambda_iλi​：不等式制約 gi(x)≤0g_i(x) \le 0gi​(x)≤0 に対応する双対変数（ラグランジュ乗数）
    • νj\nu_jνj​：等式制約 hj(x)=0h_j(x) = 0hj​(x)=0 に対応する双対変数（ラグランジュ乗数）

    です。

    2. 「双対変数がラグランジュ関数に存在する」とは？

    「双対変数がラグランジュ関数に存在する」とは、

    • 各制約ごとに専用の変数（λi,νj\lambda_i, \nu_jλi​,νj​） を用意し、
    • それらをラグランジュ関数に明示的に含める

    という意味です。

    逆に「存在しない」とすると、

    • ラグランジュ関数に双対変数を含めない
    • つまり、制約をペナルティ項などでしか扱わない、あるいは単に制約を無視する

    といった状況を指します。

    3. 双対変数がラグランジュ関数に「存在する」場合の影響

    (1) 双対問題が定義できる

    双対変数 λ,ν\lambda, \nuλ,ν を明示的に含めたラグランジュ関数 L(x,λ,ν)L(x,\lambda,\nu)L(x,λ,ν) から、

    • 主問題：min⁡xmax⁡λ≥0,νL(x,λ,ν)\min_x \max_{\lambda \ge 0, \nu} L(x,\lambda,\nu)minx​maxλ≥0,ν​L(x,λ,ν)
    • 双対問題：max⁡λ≥0,νmin⁡xL(x,λ,ν)\max_{\lambda \ge 0, \nu} \min_x L(x,\lambda,\nu)maxλ≥0,ν​minx​L(x,λ,ν)

    という双対問題が定義できます。

    これにより、

    • 強双対定理
    • 双対変数＝シャドウプライス（限界価格）
    • KKT条件による最適性の特徴づけ

    といった理論的な枠組みが使えるようになります。

    (2) 感度解析ができる

    双対変数が存在するおかげで、

    • 資源量 bbb や制約の右辺を少し変えたときの最適値の変化
    • どの制約が「タイト（有効）」で、どれが「緩い」か

    といった感度解析が可能になります。

    (3) アルゴリズム設計に役立つ

    • 双対分解（Dual decomposition）
    • ラグランジュ緩和（Lagrangian relaxation）
    • 双対上昇法（Dual ascent）

    など、双対変数を更新しながら解を求めるアルゴリズムが設計できます。

    4. 双対変数がラグランジュ関数に「存在しない」場合の影響

    (1) 双対問題やKKT条件が定義できない

    双対変数を明示的に導入していない場合、

    • 双対問題そのものが定義できません。
    • KKT条件も形式的には書けません（双対変数がないため）。

    そのため、

    • 最適解の感度
    • 制約の「価格」的な解釈
    • 双対ベースのアルゴリズム

    といった双対理論に基づく解析・手法が使えなくなります。

    (2) 代わりに使われるアプローチ

    双対変数を使わない場合、制約を扱う方法としては例えば：

    • ペナルティ法：制約違反に大きなペナルティを課す
    • バリア法：制約の内側に留まるように対数バリア項を加える
    • 射影法：各ステップで実行可能集合に射影する

    などがあります。

    これらは

    • 双対変数のような「価格」解釈は持たない
    • 感度解析は別途行う必要がある
    • 収束性や数値安定性の議論はまた別の枠組みで行う

    という違いがあります。

    5. まとめ

    • 双対変数がラグランジュ関数に存在する
      → 双対問題・KKT条件・感度解析・双対ベースのアルゴリズムなど、双対理論に基づく豊富な解析と手法が使える。

    • 双対変数がラグランジュ関数に存在しない
      → 双対理論は使えず、代わりにペナルティ法・バリア法・射影法など、主変数中心のアプローチで制約を扱うことになる。

    したがって、「双対変数をラグランジュ関数に含めるかどうか」は、理論的な枠組みと利用できるアルゴリズムの種類を大きく変える重要な選択です。

    実験

    双対変数（ラグランジュ乗数）を明示的に使う場合と使わない場合で、同じ最適化問題に対する挙動の違いが分かるように、簡単な線形計画問題を例に実装・可視化します。

    1. 問題設定（シンプルな線形計画）

    次のような利益最大化問題を考えます。

    • 活動：x1,x2x_1, x_2x1​,x2​（2種類の製品の生産量）
    • 単位利益：c1=3, c2=2c_1 = 3,\ c_2 = 2c1​=3, c2​=2
    • 資源制約：
      • 資源1：x1+x2≤4x_1 + x_2 \le 4x1​+x2​≤4
      • 資源2：x1≤3x_1 \le 3x1​≤3

    数式で書くと：

    max⁡x1,x23x1+2x2s.t.x1+x2≤4x1≤3x1≥0, x2≥0\begin{aligned} \max_{x_1,x_2} &\quad 3x_1 + 2x_2 \\ \text{s.t.} &\quad x_1 + x_2 \le 4 \\ &\quad x_1 \le 3 \\ &\quad x_1 \ge 0,\ x_2 \ge 0 \end{aligned} x1​,x2​max​s.t.​3x1​+2x2​x1​+x2​≤4x1​≤3x1​≥0, x2​≥0​

    2. 双対変数を使う場合（ラグランジュ関数＋双対上昇）

    2.1 ラグランジュ関数と双対問題

    不等式制約を

    g1(x)=x1+x2−4≤0,g2(x)=x1−3≤0g_1(x) = x_1 + x_2 - 4 \le 0,\quad g_2(x) = x_1 - 3 \le 0 g1​(x)=x1​+x2​−4≤0,g2​(x)=x1​−3≤0

    とおき、ラグランジュ関数を

    L(x,λ)=3x1+2x2+λ1(x1+x2−4)+λ2(x1−3)L(x,\lambda) = 3x_1 + 2x_2 + \lambda_1 (x_1 + x_2 - 4) + \lambda_2 (x_1 - 3) L(x,λ)=3x1​+2x2​+λ1​(x1​+x2​−4)+λ2​(x1​−3)

    と定義します（λ1,λ2≥0\lambda_1, \lambda_2 \ge 0λ1​,λ2​≥0）。

    双対問題は

    max⁡λ≥0min⁡x≥0L(x,λ)\max_{\lambda \ge 0} \min_{x \ge 0} L(x,\lambda) λ≥0max​x≥0min​L(x,λ)

    です。内側の最小化は簡単に解けます（勾配条件から xxx を λ\lambdaλ の関数として表せます）。

    2.2 双対上昇法の実装（Python）

    双対変数 λ\lambdaλ を更新しながら、双対目的関数を最大化するアルゴリズム（双対上昇法）を実装します。

    import numpy as np
import matplotlib.pyplot as plt

# 問題データ
c = np.array([3.0, 2.0])
A = np.array([[1, 1],
              [1, 0]])
b = np.array([4.0, 3.0])

# 双対上昇法（双対変数あり）
def dual_ascent(c, A, b, max_iter=100, step=0.1):
    m = A.shape[0]
    lam = np.zeros(m)  # 双対変数 λ
    history_lam = []
    history_obj = []
    
    for k in range(max_iter):
        # x(λ) = argmin_{x>=0} L(x,λ)
        # ∇_x L = c + A^T λ
        grad_x = c + A.T @ lam
        # 単純な射影: x = max(0, - (c + A^T λ)) などではなく、
        # ここでは簡略化のため線形計画ソルバを使わず、閉形式で近似
        # 実際には scipy.optimize.linprog などを使うべきですが、
        # ここでは概念を示すために簡略化します。
        x = np.maximum(0, -grad_x)  # これはあくまで概念的な近似
        
        # 双対目的値: min_x L(x,λ) の近似
        dual_obj = c @ x + lam @ (A @ x - b)
        
        history_lam.append(lam.copy())
        history_obj.append(dual_obj)
        
        # 双対変数の更新 (subgradient ascent)
        # g = A x - b が双対目的の劣勾配
        g = A @ x - b
        lam = np.maximum(0, lam + step * g)
    
    return np.array(history_lam), np.array(history_obj)

# 実行
lam_history, obj_history_dual = dual_ascent(c, A, b)

    3. 双対変数を使わない場合（単純なペナルティ法）

    同じ問題を、双対変数を使わず、制約違反にペナルティを課す方法で解きます。

    3.1 ペナルティ付き目的関数

    P(x)=3x1+2x2−μ[max⁡(0,x1+x2−4)2+max⁡(0,x1−3)2]P(x) = 3x_1 + 2x_2 - \mu \left[ \max(0, x_1 + x_2 - 4)^2 + \max(0, x_1 - 3)^2 \right] P(x)=3x1​+2x2​−μ[max(0,x1​+x2​−4)2+max(0,x1​−3)2]

    を最大化します（μ>0\mu > 0μ>0 はペナルティ係数）。

    3.2 勾配法による実装（Python）

    # ペナルティ法（双対変数なし）
def penalty_method(c, A, b, mu=10.0, max_iter=100, step=0.01):
    n = len(c)
    x = np.zeros(n)
    history_x = []
    history_obj_penalty = []
    
    for k in range(max_iter):
        # 制約違反
        violation = A @ x - b
        penalty_term = np.sum(np.maximum(0, violation)**2)
        
        # ペナルティ付き目的値
        obj_penalty = c @ x - mu * penalty_term
        history_x.append(x.copy())
        history_obj_penalty.append(obj_penalty)
        
        # 勾配 (最大化なので + 方向)
        grad = c.copy()
        for i in range(A.shape[0]):
            if violation[i] > 0:
                grad -= 2 * mu * violation[i] * A[i, :]
        x = np.maximum(0, x + step * grad)
    
    return np.array(history_x), np.array(history_obj_penalty)

# 実行
x_history, obj_history_penalty = penalty_method(c, A, b)

    5. 観察される違い（解釈）

    実行結果は以下の通りです。

    グラフの"with dual variables"が双対上昇法（双対変数あり）、"without dual varibles"がペナルティ法（双対変数なし）の結果です。

    • 双対上昇法（双対変数あり）

      • 双対変数 λ\lambdaλ が「資源の価格」として更新されていく
      • 双対目的関数値が単調に増加し、真の最適値に近づく
      • 最終的に λ\lambdaλ から「どの資源がタイトか」も読み取れる

    • ペナルティ法（双対変数なし）

      • 双対変数は存在せず、制約はペナルティ項としてのみ扱われる
      • ペナルティ係数 μ\muμ を大きくしないと制約を十分に満たさない
      • 感度（資源の限界価格）の情報は直接得られない

    このように、同じ最適化問題でも、双対変数を明示的に使うかどうかで、アルゴリズムの挙動と得られる情報（価格・感度）が大きく変わります。

    総括

    ここまでの議論をまとめます。

    • 双対変数あり：
      理論的・経済的な解釈が豊富で、大規模・分散問題に強いが、変数が増え実装はやや複雑。

    • 双対変数なし：
      問題が単純で実装が容易だが、感度情報が得られず、収束性や大規模問題への適用性で劣ることが多い。

    用途に応じて、双対変数を明示的に導入するか（双対ベースの手法を使うか）、ペナルティ法・射影法など主変数中心の手法を使うかを選択するのが一般的です。
