    テンソルを扱ったりする教科書を読むと、行列で2次の表現を得たときに、微分がパッと説明されています。

    以下の式です。

    ∂∂x(12xTQx)=12(Q+QT)x\frac{\partial}{\partial \mathbf{x}} \left( \frac{1}{2} \mathbf{x}^T Q \mathbf{x} \right) = \frac{1}{2} (Q + Q^T) \mathbf{x} ∂x∂​(21​xTQx)=21​(Q+QT)x

    これってなんでやねんと思うことが多かったなぁと思い起しました。 本日はこの変換がなぜか導出を行おうと思います。

    1次関数の微分

    y=Ax\mathbf{y} = A \mathbf{x}y=Ax というベクトルを x\mathbf{x}x で偏微分すると、結果は行列 AAA の転置になります。

    より正確には、

    ∂y∂x=AT\frac{\partial \mathbf{y}}{\partial \mathbf{x}} = A^T ∂x∂y​=AT

    です（ただし、分子レイアウトか分母レイアウトかで転置の位置が変わります）。

    1. 成分計算で確認

    x∈Rn\mathbf{x} \in \mathbb{R}^nx∈Rn、A∈Rm×nA \in \mathbb{R}^{m \times n}A∈Rm×n、y=Ax∈Rm\mathbf{y} = A \mathbf{x} \in \mathbb{R}^my=Ax∈Rm とします。

    成分で書くと、

    yi=∑j=1naijxjy_i = \sum_{j=1}^n a_{ij} x_j yi​=j=1∑n​aij​xj​

    です。

    偏微分は

    ∂yi∂xj=aij\frac{\partial y_i}{\partial x_j} = a_{ij} ∂xj​∂yi​​=aij​

    となります。

    したがって、ヤコビ行列は

    J=(∂yi∂xj)i,j=(aij)i,j=AJ = \left( \frac{\partial y_i}{\partial x_j} \right)_{i,j} = (a_{ij})_{i,j} = A J=(∂xj​∂yi​​)i,j​=(aij​)i,j​=A

    です。

    ここで、分子レイアウト（y\mathbf{y}y を縦ベクトル、x\mathbf{x}x を縦ベクトルとし、∂y∂x\frac{\partial \mathbf{y}}{\partial \mathbf{x}}∂x∂y​ を「行が y の成分、列が x の成分」と解釈する流儀）では、

    ∂y∂x=J=A\frac{\partial \mathbf{y}}{\partial \mathbf{x}} = J = A ∂x∂y​=J=A

    となります。

    一方、分母レイアウト（列が x の成分、行が y の成分）では、

    ∂y∂x=JT=AT\frac{\partial \mathbf{y}}{\partial \mathbf{x}} = J^T = A^T ∂x∂y​=JT=AT

    となります。

    最適化の文脈では、分母レイアウト（∇f=(∂f∂x1,…,∂f∂xn)T\nabla f = \left( \frac{\partial f}{\partial x_1}, \dots, \frac{\partial f}{\partial x_n} \right)^T∇f=(∂x1​∂f​,…,∂xn​∂f​)T）がよく使われるので、

    ∂(Ax)∂x=AT\frac{\partial (A \mathbf{x})}{\partial \mathbf{x}} = A^T ∂x∂(Ax)​=AT

    と書かれることが多いです。

    ヤコビ行列
    ヤコビ行列は、多変数ベクトル値関数の1階微分（勾配）を並べた行列です。

    • 関数 f(x)=(f1(x),…,fm(x))T\mathbf{f}(\mathbf{x}) = (f_1(\mathbf{x}), \dots, f_m(\mathbf{x}))^Tf(x)=(f1​(x),…,fm​(x))T に対して、

    Jf(x)=[∂f1∂x1⋯∂f1∂xn⋮⋱⋮∂fm∂x1⋯∂fm∂xn]J_{\mathbf{f}}(\mathbf{x}) = \begin{bmatrix} \frac{\partial f_1}{\partial x_1} & \cdots & \frac{\partial f_1}{\partial x_n} \\ \vdots & \ddots & \vdots \\ \frac{\partial f_m}{\partial x_1} & \cdots & \frac{\partial f_m}{\partial x_n} \end{bmatrix} Jf​(x)=​∂x1​∂f1​​⋮∂x1​∂fm​​​⋯⋱⋯​∂xn​∂f1​​⋮∂xn​∂fm​​​​

    • 幾何学的には、点 x0\mathbf{x}_0x0​ での**線形近似（接空間）**を表します。
    • 最適化では、制約関数や目的関数の勾配をまとめて扱うのに使われます。

    2. 勾配の形で見る

    スカラー関数 f(x)=aTxf(\mathbf{x}) = \mathbf{a}^T \mathbf{x}f(x)=aTx の勾配は

    ∇f=a\nabla f = \mathbf{a} ∇f=a

    です。

    同様に、ベクトル値関数 y=Ax\mathbf{y} = A \mathbf{x}y=Ax の各成分 yiy_iyi​ の勾配は

    ∇yi=(ai1,…,ain)T\nabla y_i = (a_{i1}, \dots, a_{in})^T ∇yi​=(ai1​,…,ain​)T

    です。

    これらを並べると、行列 AAA の転置 ATA^TAT になります。

    3. まとめ

    • y=Ax\mathbf{y} = A \mathbf{x}y=Ax を x\mathbf{x}x で偏微分すると、
      • 分子レイアウト：AAA
      • 分母レイアウト：ATA^TAT
    • 最適化の文脈（勾配ベクトル）では、通常分母レイアウトが使われるので、

      ∂(Ax)∂x=AT\frac{\partial (A \mathbf{x})}{\partial \mathbf{x}} = A^T ∂x∂(Ax)​=AT

      と覚えておくと便利です。

    この結果は、ラグランジュ関数の勾配条件

    ∇xL=Qx+c+ATλ\nabla_{\mathbf{x}} \mathcal{L} = Q \mathbf{x} + \mathbf{c} + A^T \boldsymbol{\lambda} ∇x​L=Qx+c+ATλ

    の中の ATλA^T \boldsymbol{\lambda}ATλ の部分を導出する際にも使われています。

    2次関数の微分

    関数

    f(x)=12xTQxf(\mathbf{x}) = \frac{1}{2} \mathbf{x}^T Q \mathbf{x} f(x)=21​xTQx

    の微分（勾配）を求めます。ここで QQQ は一般の n×nn \times nn×n 行列とします（対称とは限りません）。

    1. 行列微分の公式を使う方法

    よく知られた公式として、

    ∂∂x(xTAx)=(A+AT)x\frac{\partial}{\partial \mathbf{x}} (\mathbf{x}^T A \mathbf{x}) = (A + A^T) \mathbf{x} ∂x∂​(xTAx)=(A+AT)x

    があります。

    これを使うと、

    ∂∂x(12xTQx)=12(Q+QT)x\frac{\partial}{\partial \mathbf{x}} \left( \frac{1}{2} \mathbf{x}^T Q \mathbf{x} \right) = \frac{1}{2} (Q + Q^T) \mathbf{x} ∂x∂​(21​xTQx)=21​(Q+QT)x

    となります。

    特に QQQ が対称行列（Q=QTQ = Q^TQ=QT）なら、

    ∂∂x(12xTQx)=Qx\frac{\partial}{\partial \mathbf{x}} \left( \frac{1}{2} \mathbf{x}^T Q \mathbf{x} \right) = Q \mathbf{x} ∂x∂​(21​xTQx)=Qx

    となります。

    2. 成分計算で導出する方法

    成分ごとに計算して確認してみます。

    x=(x1,…,xn)T\mathbf{x} = (x_1, \dots, x_n)^Tx=(x1​,…,xn​)T、Q=(qij)Q = (q_{ij})Q=(qij​) とすると、

    xTQx=∑i=1n∑j=1nxiqijxj\mathbf{x}^T Q \mathbf{x} = \sum_{i=1}^n \sum_{j=1}^n x_i q_{ij} x_j xTQx=i=1∑n​j=1∑n​xi​qij​xj​

    です。

    kkk 番目の成分 xkx_kxk​ で偏微分すると、

    ∂∂xk(∑i,jxiqijxj)=∑i,j∂∂xk(xiqijxj)\frac{\partial}{\partial x_k} \left( \sum_{i,j} x_i q_{ij} x_j \right) = \sum_{i,j} \frac{\partial}{\partial x_k} (x_i q_{ij} x_j) ∂xk​∂​(i,j∑​xi​qij​xj​)=i,j∑​∂xk​∂​(xi​qij​xj​)

    となります。

    xiqijxjx_i q_{ij} x_jxi​qij​xj​ の微分は、

    • i=ki = ki=k かつ j≠kj \neq kj=k のとき：∂/∂xk(xkqkjxj)=qkjxj\partial/\partial x_k (x_k q_{kj} x_j) = q_{kj} x_j∂/∂xk​(xk​qkj​xj​)=qkj​xj​
    • i≠ki \neq ki=k かつ j=kj = kj=k のとき：∂/∂xk(xiqikxk)=xiqik\partial/\partial x_k (x_i q_{ik} x_k) = x_i q_{ik}∂/∂xk​(xi​qik​xk​)=xi​qik​
    • i=ki = ki=k かつ j=kj = kj=k のとき：∂/∂xk(xkqkkxk)=2qkkxk\partial/\partial x_k (x_k q_{kk} x_k) = 2 q_{kk} x_k∂/∂xk​(xk​qkk​xk​)=2qkk​xk​

    これらをまとめると、

    ∂∂xk(xTQx)=∑jqkjxj+∑ixiqik=(Qx)k+(xTQ)k\frac{\partial}{\partial x_k} (\mathbf{x}^T Q \mathbf{x}) = \sum_{j} q_{kj} x_j + \sum_{i} x_i q_{ik} = (Q \mathbf{x})_k + (\mathbf{x}^T Q)_k ∂xk​∂​(xTQx)=j∑​qkj​xj​+i∑​xi​qik​=(Qx)k​+(xTQ)k​

    となります。

    ベクトルとして書けば、

    ∂∂x(xTQx)=Qx+QTx=(Q+QT)x\frac{\partial}{\partial \mathbf{x}} (\mathbf{x}^T Q \mathbf{x}) = Q \mathbf{x} + Q^T \mathbf{x} = (Q + Q^T) \mathbf{x} ∂x∂​(xTQx)=Qx+QTx=(Q+QT)x

    です。

    したがって、

    ∂∂x(12xTQx)=12(Q+QT)x\frac{\partial}{\partial \mathbf{x}} \left( \frac{1}{2} \mathbf{x}^T Q \mathbf{x} \right) = \frac{1}{2} (Q + Q^T) \mathbf{x} ∂x∂​(21​xTQx)=21​(Q+QT)x

    となります。

    まとめ

    • 一般の QQQ に対して：

      ∇(12xTQx)=12(Q+QT)x\nabla \left( \frac{1}{2} \mathbf{x}^T Q \mathbf{x} \right) = \frac{1}{2} (Q + Q^T) \mathbf{x} ∇(21​xTQx)=21​(Q+QT)x

    • QQQ が対称なら：

      ∇(12xTQx)=Qx\nabla \left( \frac{1}{2} \mathbf{x}^T Q \mathbf{x} \right) = Q \mathbf{x} ∇(21​xTQx)=Qx

    これが、2次計画問題のラグランジュ関数の勾配条件に出てくる項の導出です。
